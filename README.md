# READ ME





### 프로젝트 실행

```
$npm i
$npm run start
```



### 사용한 기술

- Styled-Components 
  - 필요 시 컴포넌트의 props의 참조가 가능하기 때문에 스타일을 다르게 표현할 수 있음
  - 커스터마이징이 손쉽게 가능
